サーバーへの参加方法

Java Edition

  1. マインクラフトJava版を起動します。
  2. メインメニューから「マルチプレイ」を選択します。
  3. 「サーバーを追加」をクリックします。
  4. 以下の情報を入力します:
    • サーバー名: 私のサーバー (任意)
    • サーバーアドレス: play.myserver.com (例)
  5. 「完了」をクリックし、サーバーリストから選択して参加します。

Bedrock Edition

  1. マインクラフトBedrock版（スマホ、PC、コンソールなど）を起動します。
  2. メインメニューから「プレイ」を選択します。
  3. 「サーバー」タブに移動し、「サーバーを追加」をクリックします。
  4. 以下の情報を入力します:
    • サーバー名: 私のサーバー (任意)
    • サーバーアドレス: play.myserver.com (例)
    • ポート: 19132 (デフォルト)
  5. 「保存」をクリックし、サーバーリストから選択して参加します。
